# Example brief — batch demo

Two shots showing the shot-table format. Edit or copy this file;
the /briefs page parses any markdown table with these five columns.
`count` = number of images for image models, seconds for video models.
Every image renders on the top model (GPT Image 2 at high) — no cheap tier.

| label | model | prompt | ratio | count |
|---|---|---|---|---|
| open-wide | openai/gpt-image-2 | wide establishing shot of an empty film studio, volumetric light | 16:9 | 1 |
| close-detail | openai/gpt-image-2 | macro shot of a clapperboard slate, shallow depth of field | 1:1 | 1 |
